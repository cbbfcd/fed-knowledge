GC

GC

JS 可以直接访问保存在栈中的值，而对象的堆空间却不能直接访问，只能操作其对应的保存在栈中的引用（指针、地址）。当让该引用 = null 的时候，就可能会在下一次 GC 的时候释放该空间。

上面之所以说可能会在下一次回收，就是因为确定一块内存不再使用了是很复杂的。如果没有使用了，但是也没有释放该空间，就造成了内存泄漏。
